# URL Shortener

Сервис для сокращения ссылок на Go. Позволяет превращать длинные URL в короткие алиасы и автоматически перенаправляет по ним. Подходит для использования в личных проектах или как небольшой внутренний сервис.

---

## 📋 Что делает

- Принимает длинную ссылку и возвращает короткий алиас (можно указать свой или получить автоматически сгенерированный)
- При переходе по короткой ссылке автоматически перенаправляет на исходный URL
- Хранит данные в файловой базе данных (всё внутри одного файла, не нужен отдельный сервер БД)
- Поддерживает разные окружения: локальное, dev, продакшен
- При желании может быть развёрнут на сервере (уже есть GitHub Actions конфигурация)

---

## 🛠 Требования

- **Go** 1.20 или выше

---

## 🚀 Локальный запуск (для тестов)

```bash
# 1. Клонируем репозиторий
git clone https://github.com/PhosFactum/url-shortener.git
cd url-shortener

# 2. Скачиваем зависимости
go mod download

# 3. Запускаем сервер
go run ./cmd/url-shortener
